# CONTENT_RULES.md

**Status: CONTRACT**

## Tone & Language
All generated text must be professional, factual, and direct.

### Hard Bans (Never use these phrases)
- "why it matters"
- "robust"
- "impactful"
- "delve"
- "at the heart of"
- "unleash"
- "leverage"
- "tapestry"
- "synergy"
- "passion" (unless direct user quote)

### Style Constraints
- **Regionality:** UK English only (e.g., "organise" not "organize", "programme" not "program").
- **Readability:** Aim for a reading age of ~12–14 (KS3).
- **Sentence Structure:** Short, active sentences. Avoid nested clauses.
- **Vocabulary:** Use plain verbs (e.g., "do", "make", "show") and concrete nouns.
- **Zero Hallucination:** Never invent partners, venues, numbers, or audience metrics.

## Prohibited Outputs
- **No Marketing Tone:** No "pitching" or "selling" language. The goal is clarity, not hype.
- **No Fillers:** If the user hasn't provided details, do not speculate. State the info is missing.
