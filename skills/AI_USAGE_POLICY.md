# AI_USAGE_POLICY.md

**Status: CONTRACT**

## Permitted AI Actions
1. **Clarification:** Rewording user notes into professional UK English.
2. **Concision:** Shortening long user input to meet word counts.
3. **Structuring:** Converting a list of notes into a cohesive paragraph using the **Impact Chain** (Inputs -> Activities -> Outcomes).
4. **Tone Mapping:** Shifting user input into a "funding-safe" professional tone (as per `CONTENT_RULES.md`).
5. **Knowledge Retrieval:** Referencing the `Creative Funding Toolkit` for rate guides (Equity, BECTU, etc.) and industry standards.

## Prohibited AI Actions
1. **Fabrication:** Adding partners, venues, or artistic outcomes not mentioned by the user.
2. **Financial Guessing:** Inventing budget figures or match-funding sources.
3. **"Ambition Inflation":** Making the project sound more "ambitious" or "important" using hype words.
4. **Data Supplementation:** Using external knowledge to "flesh out" a project (e.g., looking up a venue's capacity).

## Fallback Behavior
- **Condition:** User triggers "Rewrite with AI" on a slot with sub-minimum input.
- **Response:** "I need more detail about [Topic] to draft this section. Please add [X] in the previous step."
- **Never:** Generate generic boilerplate text.
